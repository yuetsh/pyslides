## 输出

使用 `print()` 函数来把东西输出到屏幕上，例如：

```py
print("你好啊") # 文字需要用引号包裹
```

不光可以输出**文字**，也可以输出**数字**和**计算结果**等

```py
print(100)
print(3.14)
print(3.14*100-10)
```

得到结果：

```py
100
3.14
304
```
